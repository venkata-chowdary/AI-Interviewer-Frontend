# Frontend

The main project documentation now lives in the repository root:

- `../README.md`

This folder contains the Next.js application for:

- authentication
- dashboard and analytics
- resume upload
- job description matching
- interview session UI
- detailed reports
- settings and profile management

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

For complete setup, backend configuration, environment variables, and feature documentation, read the root `README.md`.
